小傅哥-提醒: Fastjson反序列化风险

攻击者可以通过此漏洞远程执行恶意代码来入侵服务器。


错误实践:

class Foo {
    void bar() {
        ParserConfig.getGlobalInstance().setAutoTypeSupport(true);
    }
}

JVM启动参数: -Dfastjson.parser.autoTypeSupport=true


最佳实践:

(1) 使用Gson组件。

(2) 禁止使用 setAutoTypeSupport(true)。

(3) 禁止使用JVM参数 -Dfastjson.parser.autoTypeSupport=true。
